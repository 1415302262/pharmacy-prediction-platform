API 调用

平台提供 JSON API，可接前端、自动化脚本或其他系统。

POST /api/predict
{
  "smiles": "Cn1c(CN2CCN(CC2)c3ccc(Cl)cc3)nc4ccccc14"
}

返回：标准化SMILES、集成预测值、解释模型预测值、分子图像、正负贡献特征。

Gemini API 调用示例
import requests

response = requests.post(
    "https://your-space-url.hf.space/api/gemini-explain",
    json={
        "smiles": "Cn1c(CN2CCN(CC2)c3ccc(Cl)cc3)nc4ccccc14",
        "focus": "药化改造"
    }
)
print(response.json())

需要在 Hugging Face Space 的 Secrets 中配置 `ECOSORT_GEMINI_API_KEY`。
